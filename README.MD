# 🧵 Bead Patterns

Bead Patterns es una aplicación web para diseñar patrones de pulseras con mostacillas (miyuki), pensada para facilitar la creación, organización y reutilización de diseños de forma fácil.

## ✨ Inspiración

Mi mamá empezó hace unos meses a hacer pulseras como hobby y para vender, utilizando mostacillas (miyuki), pero crear los diseños a mano es complicado. Las aplicaciones móviles que existen no son muy buenas ni útiles para diseñar este tipo de piezas, por lo que hasta ahora ha usado partes de diseños de internet en lugar de crear los suyos propios.

Este proyecto nace para darle una herramienta simple y potente que le permita diseñar sus propios patrones.

## ✨ Features

- 🎨 **Editor de patrones**
  - Crea telares personalizados definiendo filas y columnas  
  - Dibujo libre sobre una cuadrícula tipo mostacillas  

- 🌈 **Paleta de colores**
  - Selección y uso de colores para diseñar patrones fácilmente  

- 🧵 **Diseño de pulseras**
  - Creación visual de patrones de mostacillas (miyuki)  

- 💾 **Gestión de proyectos**
  - Crear, guardar y cargar diseños  
  - Clonar/copiar proyectos existentes para reutilizarlos  

- 🖼️ **Conversión de imágenes**
  - Convierte imágenes en patrones de mostacillas automáticamente  

- 📤 **Exportación de diseños**
  - Exporta los patrones como imágenes para compartirlos fácilmente  

- ⚡ **Aplicación ligera**
  - Construida con Vanilla TypeScript, Web Components, Vite y Tailwind  
  - Mínima dependencia de librerías externas
